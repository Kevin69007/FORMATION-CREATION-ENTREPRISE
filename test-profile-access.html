Test d'Accès au Profil

1. Test de Navigation Dashboard → Profil

Vérifiez que le lien "Mon Profil" est visible dans la sidebar du dashboard :

Ouvrir Dashboard
Instructions :
  1. Ouvrez le dashboard
  2. Vérifiez que le lien "Mon Profil" est visible dans la sidebar
  3. Cliquez sur le lien pour accéder au profil

2. Test de Navigation Profil → Dashboard

Vérifiez que le bouton "Retour au Dashboard" fonctionne :

Ouvrir Profil
Instructions :
  1. Ouvrez le profil
  2. Cliquez sur "Retour au Dashboard"
  3. Vérifiez que vous revenez au dashboard

3. Test de Persistance du Lien Profil

Le lien profil doit être visible même quand on navigue dans les modules :

Ouvrir Module 1, Leçon 1
Instructions :
  1. Ouvrez une leçon
  2. Vérifiez que le lien "Mon Profil" est toujours visible dans la sidebar
  3. Cliquez sur le lien pour accéder au profil

4. Test de Fonctionnalités du Profil

Vérifiez que les fonctionnalités du profil fonctionnent :

Ouvrir Profil
Instructions :
  1. Ouvrez le profil
  2. Vérifiez que les informations personnelles s'affichent
  3. Testez la modification des informations (prénom, nom, email)
  4. Vérifiez que la progression s'affiche correctement

5. Test de Création d'Étudiant

Testez la création d'un nouveau compte étudiant :

Ouvrir Interface Admin Test Création Étudiant
Instructions :
  1. Ouvrez l'interface admin
  2. Cliquez sur "Nouvel étudiant"
  3. Remplissez le formulaire avec des données de test
  4. Vérifiez que la création fonctionne sans erreur
